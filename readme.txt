 SplitKit v1.1.0 - README

Thank you for using SplitKit!

-------------------------------------------------
 Description
-------------------------------------------------
SplitKit is a simple desktop application designed to quickly and easily split
long video files into multiple shorter clips of a fixed duration.

 Installation / Running
-------------------------------------------------
[If using an installer:]
Run the SplitKit_Setup.exe installer and follow the on-screen instructions.
Shortcuts will be created in your Start Menu and/or Desktop.

-------------------------------------------------
 Basic Usage
-------------------------------------------------
1. Launch SplitKit.
2. Use the "Browse..." buttons to select your input video file and an output folder.
3. Set the desired "Clip Duration" in seconds.
4. Adjust optional settings like "Start Offset" or "Max Clips".
5. Choose a "Clipping Mode" (Fast Copy or Smaller Size).
6. Click "Start Clipping".
   (Refer to the application's "About" page for more detailed instructions).
